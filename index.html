Programming Logical Training

Object Oriented Programming

Advanced Java OOP: classes, objects, constructors; inheritance and method overriding; polymorphism (compile-time and runtime); interfaces and abstract classes; encapsulation and access modifiers; exception handling (try/catch/finally, custom exceptions); generics and type safety; Java Collections Framework (ArrayList, HashMap, LinkedList, TreeSet); iterators; introductory design patterns (Singleton, Factory, Observer).

Elementary Statistics

Data types and measurement scales; descriptive statistics — mean, median, mode, variance, standard deviation; frequency distributions and histograms; probability rules (addition, multiplication, conditional); Bayes' theorem; discrete distributions (Binomial, Poisson); continuous distributions (Normal, t-distribution); Central Limit Theorem; confidence intervals; hypothesis testing (z-test, t-test, chi-square); correlation and simple linear regression — personally one of my favourite courses.

Cybersecurity Management

Human Factors in Digital Interaction Design
